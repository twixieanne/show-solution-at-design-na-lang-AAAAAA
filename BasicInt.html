Fundamental Integration Rules

These are the building blocks of integral calculus. Memorizing these basic forms will help you solve more complex problems.

Video credit: The Organic Chemistry Tutor. All rights reserved.

Function Integral Example
∫ k dx kx + C ∫ 5 dx = 5x + C
∫ xⁿ dx (n ≠ -1) xⁿ⁺¹/(n+1) + C ∫ x³ dx = x⁴/4 + C
∫ eˣ dx eˣ + C ∫ eˣ dx = eˣ + C
∫ 1/x dx ln|x| + C ∫ 3/x dx = 3ln|x| + C
∫ sin(x) dx -cos(x) + C ∫ sin(x) dx = -cos(x) + C
∫ cos(x) dx sin(x) + C ∫ cos(x) dx = sin(x) + C

Practice Examples

Example 1: Power Rule

Find ∫ x⁵ dx

Using the power rule:

∫ xⁿ dx = xⁿ⁺¹/(n+1) + C

∫ x⁵ dx = x⁶/6 + C

Example 2: Constant Multiple

Find ∫ 4eˣ dx

∫ k·f(x) dx = k·∫ f(x) dx

∫ 4eˣ dx = 4∫ eˣ dx

= 4eˣ + C

Example 3: Sum Rule

Find ∫ (3x² + 2x + 1) dx

∫ [f(x) + g(x)] dx = ∫ f(x) dx + ∫ g(x) dx

∫ (3x² + 2x + 1) dx = 3∫ x² dx + 2∫ x dx + ∫ 1 dx

= 3(x³/3) + 2(x²/2) + x + C

= x³ + x² + x + C
