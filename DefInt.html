Definite Integrals

← Previous Next →

Fundamental Concept

Definite integrals calculate the net area under a curve between two points on the x-axis.

∫ab f(x) dx = F(b) - F(a)

where F is the antiderivative of f

Properties of Definite Integrals

Property Expression
Reversed Limits ∫ab f(x) dx = -∫ba f(x) dx
Zero-Length Interval ∫aa f(x) dx = 0
Additivity ∫ab f(x) dx + ∫bc f(x) dx = ∫ac f(x) dx
Constant Multiple ∫ab k·f(x) dx = k·∫ab f(x) dx
Sum Rule ∫ab [f(x) + g(x)] dx = ∫ab f(x) dx + ∫ab g(x) dx

The Fundamental Theorem of Calculus

If f is continuous on [a, b] and F is an antiderivative of f, then:

∫ab f(x) dx = F(b) - F(a)

This theorem connects differential and integral calculus and provides a way to evaluate definite integrals.

Worked Examples

Example 1: Basic Definite Integral

Evaluate ∫13 (2x + 1) dx

Find the antiderivative:

F(x) = x² + x + C

Apply the FTC:

F(3) - F(1) = (3² + 3) - (1² + 1) = (9 + 3) - (1 + 1) = 12 - 2 = 10

Example 2: Definite Integral with Substitution

Evaluate ∫01 x ex² dx

Let u = x² ⇒ du = 2x dx ⇒ x dx = du/2

Change limits: when x=0, u=0; when x=1, u=1

Substitute:

∫01 eu (du/2) = ½ eu |01 = ½(e - 1)

Example 3: Absolute Value Function

Evaluate ∫-23 |x| dx

Split at x=0 where behavior changes:

∫-23 |x| dx = ∫-20 -x dx + ∫03 x dx

= -½x² |-20 + ½x² |03

= [0 - (-½(4))] + [½(9) - 0] = 2 + 4.5 = 6.5

Geometric Interpretation

Definite integral represents the net area between the curve and x-axis

  • Areas above x-axis are positive
  • Areas below x-axis are negative
