The Integration by Parts Formula

This method is derived from the product rule of differentiation and is useful for integrals involving products of functions.

∫ u dv = uv - ∫ v du

How to Apply the Method:

  1. Identify parts of the integrand to set as u and dv (LIATE rule can help)
  2. Differentiate u to find du
  3. Integrate dv to find v
  4. Apply the integration by parts formula
  5. Simplify and solve the resulting integral

LIATE Rule (Priority order for choosing u):

  • Logarithmic functions (ln x)
  • Inverse trigonometric functions (arctan x)
  • Algebraic functions (xⁿ)
  • Trigonometric functions (sin x)
  • Exponential functions (eˣ)

Worked Examples

Example 1: Basic Application

Find ∫ x eˣ dx

Let u = x ⇒ du = dx

Let dv = eˣ dx ⇒ v = eˣ

Apply the formula:

∫ x eˣ dx = uv - ∫ v du = x eˣ - ∫ eˣ dx

= x eˣ - eˣ + C = eˣ(x - 1) + C

Example 2: Logarithmic Function

Find ∫ ln x dx

Let u = ln x ⇒ du = (1/x) dx

Let dv = dx ⇒ v = x

Apply the formula:

∫ ln x dx = x ln x - ∫ x (1/x) dx

= x ln x - ∫ 1 dx = x ln x - x + C

Example 3: Repeated Application

Find ∫ x² sin x dx

First application:

Let u = x² ⇒ du = 2x dx

Let dv = sin x dx ⇒ v = -cos x

∫ x² sin x dx = -x² cos x + ∫ 2x cos x dx

Second application (on the new integral):

Let u = 2x ⇒ du = 2 dx

Let dv = cos x dx ⇒ v = sin x

∫ 2x cos x dx = 2x sin x - ∫ 2 sin x dx

= 2x sin x + 2 cos x + C

Final answer:

-x² cos x + 2x sin x + 2 cos x + C
